# Build from source

```bash
git clone https://github.com/cdbkk/claude-menubar
cd claude-menubar
./build.sh            # builds build/ClaudeMenuBar.app
./build.sh --dmg      # also produces build/ClaudeMenuBar.dmg
```
Requires the Xcode Command Line Tools (`xcode-select --install`).
